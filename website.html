Merkzettel: IT-Dienste - Datensicherung

1. Was ist Datensicherung?

Datensicherung, auch Backup genannt, bezeichnet das Erstellen von Kopien wichtiger Daten, um diese im Falle von Datenverlust wiederherstellen zu können.

2. Gründe für Datensicherung

  • Schutz vor Hardware-Fehlern
  • Schutz vor Malware und Ransomware
  • Versehentliches Löschen von Dateien
  • Diebstahl oder Verlust von Geräten
  • Katastrophen wie Feuer oder Überschwemmungen

3. Arten von Backups

  • Vollbackup: Sichert alle Daten komplett.
  • Inkrementelles Backup: Sichert nur die seit dem letzten Backup geänderten Daten.
  • Differenzielles Backup: Sichert alle Änderungen seit dem letzten Vollbackup.
  • Spiegelung (Mirroring): Echtzeitkopie der Daten auf ein anderes Medium.

4. Backup-Strategien

Eine bewährte Strategie ist die 3-2-1-Regel:

  • 3 Kopien der Daten
  • 2 verschiedene Speichermedien
  • 1 Kopie an einem externen Standort

Eine weitere Strategie ist das Großvater-Vater-Sohn-Prinzip:

  • Großvater: Monatliche Backups
  • Vater: Wöchentliche Backups
  • Sohn: Tägliche Backups

5. Speicherorte für Backups

  • Externe Festplatten
  • Netzwerkspeicher (NAS)
  • Cloud-Dienste
  • Optische Medien (z.B. DVDs)
  • USB-Sticks (nur für kurzfristige Backups empfohlen)
  • Bandlaufwerke (für große Datenmengen und Langzeitarchivierung)

6. Wichtige Aspekte der Datensicherung

  • Regelmäßigkeit: Backups sollten in festgelegten Intervallen erstellt werden.
  • Automatisierung: Automatisierte Backups reduzieren menschliche Fehler.
  • Verschlüsselung: Schutz sensibler Daten durch Verschlüsselung.
  • Testen der Backups: Regelmäßige Wiederherstellungstests stellen die Funktionsfähigkeit sicher.
  • Versionierung: Mehrere Versionen von Dateien speichern, um ältere Versionen bei Bedarf wiederherstellen zu können.
  • Sicherer Aufbewahrungsort: Backups sollten vor unbefugtem Zugriff geschützt werden.

7. Wiederherstellung von Daten

Der Wiederherstellungsprozess (Restore) ist genauso wichtig wie das Backup selbst. Ohne erfolgreiche Wiederherstellung ist ein Backup wertlos. Es gibt:

  • Vollständige Wiederherstellung: Das gesamte System wird wiederhergestellt.
  • Partielle Wiederherstellung: Nur ausgewählte Dateien oder Ordner werden wiederhergestellt.
  • Bare-Metal-Recovery: Wiederherstellung auf neuer Hardware ohne vorheriges Betriebssystem.

8. Tools und Software für Backups

  • Acronis True Image
  • Veeam Backup & Replication
  • Windows Backup
  • Time Machine (für Mac)
  • Cloud-Dienste wie Google Drive, Dropbox, OneDrive
  • Backup Exec (Symantec)
  • rsync (Linux/Unix-basierte Systeme)
